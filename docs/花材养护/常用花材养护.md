# 玫瑰养护

正面（核心步骤）
┌─────────────────────────┐
│ │
│ 🌹 玫瑰养护 │
│ （花已醒好·直接瓶插） │
│ │
│ ① 水位 10-20cm │
│ （花瓶约1/3高） │
│ │
│ ② 每500ml自来水加： │
│ ¼ 二氧化氯 + ½ 保鲜剂 │
│ │
│ ③ 一水到底 ❌不换水 │
│ 只加同比例的水 │
│ │
│ ④ 远离空调出风口 │
│ 远离成熟水果 │
│ │
│ 🌿 花头不喷水 │
│ 水下不留叶 │
│ │
└─────────────────────────┘

反面（急救 + 附赠品保存）
┌─────────────────────────┐
│ │
│ ⚠️ 唯一例外： │
│ 水质变浑浊时 │
│ → 重新配水（同比例） │
│ → 根部斜切45° │
│ → 切口越大越好 │
│ │
│ 📦 附赠品保存 │
│ • 二氧化氯片（余¾片） │
│ 干燥密封，下次用¼ │
│ • 保鲜剂（余½包） │
│ 折口密封，下次用½ │

# 百合养护

正面（核心步骤）

┌─────────────────────────┐
│ │
│ 🌸 百合养护 │
│ （花已醒好·直接瓶插） │
│ │
│ ① 水位 15-20cm │
│ （花瓶约1/2高） │
│ │
│ ② 每500ml自来水加： │
│ ¼ 二氧化氯 + ½ 保鲜剂 │
│ │
│ ③ 一水到底 ❌不换水 │
│ 只加同比例的水 │
│ │
│ ④ 远离空调出风口 │
│ 远离成熟水果 │
│ │
│ 🌿 水下不留叶 │
│ 花头不喷水 │
│ │
└─────────────────────────┘

反面（花粉处理 + 急救 + 附赠品保存）

┌─────────────────────────┐
│ │
│ ✨ 百合特别小贴士 │
│ • 花苞张开后，立即 │
│ 摘掉**雄蕊**（带花粉的） │
│ 可用纸巾包着拔掉 │
│ • 花粉沾到衣服/皮肤： │
│ 用胶带粘掉，不要用水搓 │
│ │
│ ⚠️ 唯一例外： │
